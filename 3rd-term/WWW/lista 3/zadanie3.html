Wszystkie div mają ustawione top: 200px; left: 100px;

Static sprawia, że top i left nie są uwzględniane.

Relative sprawia, że element jest pozycjonowany względem pozycji, którą posiadałby normalnie bez top i left. Obszar, który normalnie zajmowałby element nie jest "uwalniany".

Absolute sprawia, że element jest przesuwany względem krawędzi paddingu elementu pierwszego elementu obejmującego który nie jest static.

Fixed sprawia, że element jest pozycjonowany względem krawędzi okna przeglądarki, również jeśli treść zostanie przewinięta.
